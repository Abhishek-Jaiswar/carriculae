## Application for building carriculams
### System design

This application aims to provide users a carriculam tracking system.

For example, a user can create a carriculam for a subject, and then add topics to it. Or use ai to generate a carriculam for a subject based on a few keywords.

Here is the components,
***
1. A page where all the subjects will be listed in cards.
2. A page where a user can create a carriculam for a subject.
3. A page where a user can add topics to a carriculam.
4. A page where a user can generate a carriculam for a subject using ai.
5. A page where a user can track the progress of a carriculam.
6. Carriculam details page includes 
    * A kanban like structure with chekboxes something like that with progress bar


I have setup a basic nextjs application with tailwindcss and shadcn ui. You can find the code in the `carriculam` folder.


I want you to build the application based on the system design above.
